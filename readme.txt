I. Состав утилиты.

1. imp1c_edo.exe - утилита импорта данных из 1С
2. imp1c_edo.exe.config - файл настроек утилиты
3. etl.dll - библиотека основных функций
При первом запуске утилита создаёт в каталоге своего запуска каталог "logs".
В этот каталог утилита пишет свои логи в текстовом формате.

II. Функциональность.

Утилита выполняет импорт данных из конфигурации "1С:Документооборот"
в базу данных MS SQL Server. Для подключения к 1С используется ComConnector.
Выполняется импорт следующих справочников:
1. Внутренние документы.
2. Файлы. Импортируются только те записи, владельцами которых являются
элементы справочника "Внутренние документы".

Импорт справочника "Внутренние документы" (50 тыс. записей) и
"Файлы" (115 тыс. записей) выполняется приблизительно 30 минут.

III. Требования к установке.

Требуется установка .NET Framework 4.5.
